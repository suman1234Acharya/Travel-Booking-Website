# WanderVibe - Travel Booking Website

A fully responsive travel booking website with hotel and flight search, special deals, destination inspiration, and a contact form. Built with HTML, CSS, and JavaScript.

## Features
- Hotel & Flight search (5 each)
- Dynamic results filtering
- Deal cards with discounted prices
- Trending destinations gallery
- Contact form with validation
- Smooth page navigation (single-page app style)

## How to Use
1. Clone or download this repository.
2. Open `index.html` in any modern browser.
3. No server or installation required.

## File Structure
- `index.html` - main markup
- `style.css` - all styles
- `script.js` - application logic
- `README.md` - project info

## Demo Data
All data is mocked for demonstration purposes (hotels, flights, deals, destinations). Images are from Lorem Picsum.
